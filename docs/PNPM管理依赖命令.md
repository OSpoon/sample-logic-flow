# PNPM 管理依赖命令

| 命令              |
| ----------------- |
| pnpm add          |
| pnpm install      |
| pnpm update       |
| pnpm remove       |
| pnpm link         |
| pnpm unlink       |
| pnpm import       |
| pnpm rebuild      |
| pnpm prune        |
| pnpm fetch        |
| pnpm install-test |
| pnpm dedupe       |

1. -w: 安装到 workspace root
2. --filter: 安装到指定的 package
3. -D: 安装到 devDependencies
4. -O: 安装到 optionalDependencies
